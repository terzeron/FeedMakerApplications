코드업1469 2차원 지그재그 채우기

입력값 n에 대해 주어진 패턴대로 2차원 리스트를 생성하고 출력하는 문제입니다.

규칙 분석
짝수행(인덱스 0, 2, 4, ...) → 오른쪽에서 왼쪽
홀수행(인덱스 1, 3, 5, ...) → 왼쪽에서 오른쪽
으로 숫자 채우기

n = int(input())
data = [[0]*n for _ in range(n)] # 리스트 0으로 초기화

num = 1 # 시작 숫자
for i in range(n):
    if i%2==1: # 홀수행은 왼쪽에서 오른쪽으로 채우기
        for j in range(n):
            data[i][j] = num
            num +=1
    else:      # 짝수행은 오른쪽에서 왼쪽으로 채우기
        for j in range(n-1,-1,-1):
            data[i][j] = num
            num +=1

for i in data:
    print(*i)

코드업1477 2차원 배열 빗금 채우기

주어진 n × m 크기의 2차원 배열을 대각선 방향으로 증가하는 숫자로 채우는 문제입니다.
대각선을 따라 숫자를 채웁니다 (d는 0부터 n + m - 2까지 반복).
각 대각선의 시작 위치를 계산합니다:
col = min(d, m - 1) → 대각선의 시작 열(오른쪽 끝을 넘지 않도록 설정)
row = d - col → 대각선의 시작 행을 조정

대각선 (↙ 아래에서 위로) row 를 증가시키고, col 을 줄이면서 숫자를 채웁니다. 따라서, 행(row)과 열(col)의 합은 항상 대각선 인덱스 d와 같습니다. ( row = d - col 인 이유)

조건을 만족할 때까지 반복하며 숫자를 채웁니다:
col이 배열 범위 안에 있어야 합니다 (col >= 0).
row가 배열 범위 안에 있어야 합니다 (row < n).

n, m = map(int, input().split())
data = [[0]*m for _ in range(n)] # n X m 리스트 초기화

num=1 # 채울 숫자

for d in range(n + m - 1):  # 대각선 개수만큼 반복
    col = min(d, m - 1) # 대각선 시작 위치: col가 m-1을 넘으면 인됨 (오른쪽 끝을 넘지 않도록 설정)
    row = d - col  # 시작 행 조정
    while col >= 0 and row < n:
        data[row][col] = num
        num += 1
        row += 1
        col -= 1

for i in data:
    print(*i)

코드업1484 2차원 배열 달팽이 채우기

주어진 n × m 크기의 2차원 배열을 나선형(Spiral) 형태로 채우는 문제입니다.
숫자는 시계 방향(→ ↓ ← ↑) 순서로 이동하며 채워진다.
벽이나 이미 숫자가 채워진 칸을 만나면 방향을 바꾼다.

2D 리스트는 data[row][col] 형태로
row (Y축) 값이 증가하면 아래로 이동,
col (X축) 값이 증가하면 오른쪽으로 이동합니다.

directions 리스트를 사용하여 우(→), 하(↓), 좌(←), 상(↑) 순서로 이동합니다.
숫자를 채우고 나면 다음 방향으로 이동, 벽을 만나거나 이미 채워진 칸을 만나면 방향을 변경합니다.

n, m = map(int, input().split())
data = [[0]*m for _ in range(n)] # n X m 리스트 초기화

num=1 # 채울 숫자

directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]  # → ↓ ← ↑  (시계방향 순서)
x, y, d = 0, 0, 0  # 시작 위치 및 방향

for _ in range(n * m):
    data[y][x] = num
    num += 1
    nx, ny = x + directions[d][1], y + directions[d][0] # 다음 위치 계산

    # 벽을 만나거나 이미 값이 채워진 경우, 방향 전환
    if not (0 <= nx < m and 0 <= ny < n and data[ny][nx] == 0):
        d = (d + 1) % 4  # 다음 방향으로 변경
        nx, ny = x + directions[d][1], y + directions[d][0]

    x, y = nx, ny  # 새로운 위치 이동

for i in data:
    print(*i)

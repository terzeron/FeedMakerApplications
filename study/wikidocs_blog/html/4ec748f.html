백준 1103번 게임

@angie
2024년 11월 19일 2:52 오전

이 문제는 게임 보드 위에서 동전을 움직이며 최대한 많은 횟수를 기록하는 문제입니다.
보드의 각 칸에는 1부터 9까지의 숫자가 적혀 있으며, 해당 숫자만큼 상하좌우 방향으로 이동해야 합니다.
보드의 일부 칸은 'H'로 표시된 구멍이고, 동전이 이 칸이나 보드 바깥으로 나가면 게임이 종료됩니다.

풀이

  • 이 문제는 풀기 위해서는 보드의 모든 가능한 이동 경로를 탐색하면서 중복 계산을 피하고, 무한 루프를 방지해야 합니다.
  • 이를 위해서 깊이 우선 탐색(DFS), 메모이제이션, 그리고 백트래킹을 사용할 수 있습니다.
  • DFS 탐색: 현재 위치에서 이동 가능한 모든 경로를 재귀적으로 탐색합니다.
  • 메모이제이션: 탐색 중 계산된 각 칸의 최대 이동 횟수를 저장해, 동일한 칸에 대해 중복 계산을 피합니다.
  • 백트래킹: 방문한 경로를 체크하고, 탐색이 끝나면 방문 여부를 다시 해제하여 다른 경로를 탐색할 수 있도록 합니다.
  • 무한 루프 방지: 이미 방문했던 칸을 다시 방문할 경우 무한 루프가 발생할 수 있으므로, 이를 감지하면 -1을 출력하고 프로그램을 종료합니다.

Python 코드

import sys
sys.setrecursionlimit(10**6)
input = sys.stdin.readline

n,m = map(int, input().split())
board = [list(input().strip()) for _ in range(n)]

dp = [[0]*m for _ in range(n)]
visited = [[0]*m for _ in range(n)]

def in_range(x,y):
    return 0<=x<n and 0<=y<m

def dfs(x,y):
    if dp[x][y]:      # 이미 계산된 칸이면 바로 반환
        return dp[x][y]

    if visited[x][y]: # 무한 루프 감지
        print(-1)
        exit()

    visited[x][y] = 1 # 현재 칸 방문 표시
    max_move = 1
    k = int(board[x][y]) # 이동해야 하는 거리
    for dx,dy in [(0, 1), (0, -1), (1, 0), (-1, 0)]:
        nx,ny = x+dx*k, y+dy*k
        if not in_range(nx,ny): # 보드를 벗어나면 0
            continue
        if board[nx][ny] == "H": # 구멍에 빠지면 0
            continue
        max_move = max(max_move, dfs(nx,ny)+1) # 이동 횟수 업데이트

    visited[x][y] = 0 # 백트래킹: 현재 칸 방문 해제
    dp[x][y] = max_move # 현재 칸까지의 최대 이동 횟수
    return dp[x][y]

print(dfs(0,0))

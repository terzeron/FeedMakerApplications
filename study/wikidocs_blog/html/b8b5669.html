ipTIME NAS의 Piwigo Upgrade하기

@british
2026년 1월 10일 8:01 오후

ipTIME의 NAS2Dual 을 사용하고 있다. plugin App으로 Piwigo 2.10.2 버전이 제공된다. (아주 오래된 버전이다) 좀 더 최신 버전을 설치하고자 하여도 ipTIME에서는 기본 제공되는 버전외의 Upgrade 등을 지원하지 않는다.

현재 NAS에 설치된 APM (Apache, PHP, MySQL)에서 PHP 버전은 7.3.17 버전이 설치되어 있다. APM은 firmware update로 인해 한번 Update된 상태이며, 더이상 버전업을 기대하긴 힘든다.

Piwigo는 16.0 버전부터 PHP 7.4 이상을 요구하고 있으므로 15.x 버전까지 사용가능하다. (이론상...)
15.x의 마지막 버전은 15.7.0으로 확인된다. piwigo-15.7.0 버전은 download하여 미리 준비해 놓는다.

ipTIME NAS에서 Piwigo를 설치하는 방법은 인터넷에 많이 공개되어 있으므로 생략한다.

일단, Plugin App에서 piwigo를 설치한다.
나의 경우 plugin/apache/piwigo210 에 설치가 되었다. piwigo 설정으로 이동해서 install 진행하지 말고, plugin/apache/piwigo210에 ftp로 접근하여 설치된 파일을 모두 지운다. 없어도 되지만, 아래 파일은 남겨 둔다.

en_help.html
ko_help.html
piwigo210.png

plugin/apache/piwigo210에 piwigo-15.7.0 버전의 압축을 푼 파일들을 모두 복사한다. 그런 후 piwigo page (http://myipdiskURL:8000/apps/piwigo210)로 접속하면 15.7.0 버전의 설치 화면이 나타난다. 이제부터 15.7.0 버전의 piwigo를 사용하면 된다.

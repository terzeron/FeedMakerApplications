문제링크 : https://www.acmicpc.net/problem/9375

해당 문제는 Hashmap 으로 풀면 되는데, Hashmap이란

HashMap<String, Integer> fashionmap = new HashMap<>();
String이 Key, Integer가 Value이다. 각 Key와 Value가 하나의 묶음처럼 저장되어서
아주 낮은 시간 복잡도를 자랑한다.

  String item = st.nextToken();
  String Section = st.nextToken();

  fashionmap.put(Section, fashionmap.getOrDefault(Section, 0) + 1);

다른건 중요치 않다. 뒤에나오는 문자열인 옷의 종류가 중요함. ( 이걸 기준으로 옷 조합을 세기 때문)
fashionmap.put(Section, fashionmap.getOrDefault(Section, 0) + 1);
처음엔 이 코드가 정말 헷갈렸는데, fashionmap에 Section을 Key값으로 두고, 그 뒤에 Key가 이미 있으면 +1을, 없으면 0 을 주는 코드이다. 해시맵 사용할때 자주 사용하고, 정말 치트키같은 코드니까 꼭 기억 해둬야한다..

패션의 합계는 다음과 같이 구하면 되는데, 패션의 종류를 모두 입거나, 하나씩 줄여가며 입거나, 아무것도 입지않거나 같은 선택지가 있다 이중 '아무것도 입지않거나' 때문에 +1을 해주며,

최종 출력값은 -1을 해줘야한다(그 어떤것도 착용하지않은 알몸상태는 제외한다고 문제에 써져있기 때문이다)

[pyhwpx QnA] hwp.insert_file로 문서병합이 되지 않는다고요?

@ilco
2024년 12월 23일 6:49 오후
Fileclose Movedocbegin Copypage Tabclose Pagecount Pastepage Insert_file Goto_page Switch_to Add_tab Close Add_doc Movedocend 병합 Qna Pyhwpx

아래아한글 자동화를 체험해보실 수 있도록, 샘플 문서와 간단한 파이썬 코드를 제공해드립니다.

실습용 샘플문서 다운로드

며칠 전 인프런에서 한 제보를 받았습니다.

용지여백이나 스타일이 다양한 문서들을 한 개의 파일에 병합하고 싶은데

"문서 끼워넣기" 기능을 수행하는 insert_file 메서드가 제대로 작동하지 않는다는 내용이었습니다.

원래는 병합한 문서들마다 여백이나 스타일이 제각각이어야 하는데

insert_file로 병합한 후에 확인해보니

첫 번째 문서의 여백대로 다른 모든 문서들이 바뀌어버려서 서식이 다 깨진다고 하셨어요.

(샘플문서들입니다. 미묘하게 네 개 문서들의 여백이 다른데, 병합하면 똑같아져버린다고...)

어? 그럴 리가 없는데?

얼른 제 PC에서 테스트를 해보았습니다.

....잘 되는데?

import os  # 경로와 파일 다루기 위해 os 임포트
from pyhwpx import Hwp  # 아래아한글 조작하기 위해 pyhwpx.Hwp 임포트

hwp_list = [i for i in os.listdir() if i.endswith(".hwp")]  # 문서 목록을 리스트로 만들어놓기

hwp = Hwp()  # 아래아한글을 실행한 후
hwp.open(hwp_list[0])  # 첫 번째 문서를 열고
hwp.MoveDocEnd()  # 문서 끝으로 이동한 후에
for h in hwp_list[1:]:  # 다른 문서들을 하나씩
    hwp.insert_file(h, move_doc_end=True)  # 첫 번째 문서 끝에 끼워넣기한 후, 문서 끝으로 이동 반복

...이상하게도 제 PC에서는 잘 병합이 됩니다ㅜ

종종 이런 경우들이 있었습니다.

이유는 모르지만 내 PC에선 잘 되고, 고객이나 수강생 분들 PC에서 안 되는..

이런 경우가 가장 해결이 까다롭습니다.

우선 오류의 원인 파악이 어려운데다, 익숙한 방법 대신 새 방법을 고민해야 하기 때문입니다.

다행히 대안이 있다!

그래도 다행히 insert_file 메서드는 믿음직한 대안이 있습니다.

바로 쪽 복사(CopyPage)와 쪽 붙여넣기(PastePage) 액션입니다.

한/글2014 이하의 버전에는 위 기능(쪽복사, 쪽붙이기)이 없습니다.
(VP 이후 버전에서만 사용 가능합니다.)
막상 장점을 설명드리자니 좀 막연하지만, 하여튼 쪽 안에 있는 모든 걸 어떤 흐트러짐도 없이 복붙해줍니다.
저를 여러 번 살려준 고마운 액션이기도 합니다.

출근하긴 했지만, 질문 주신 분이 긴박한 상황이신 듯 하여

오전중에 몰래 아래아한글에 휴먼명조로 코드를 작성하고,

후다닥 vscode로 옮겨 테스트만 한 번 해본 후 메일로 코드를 보내드렸습니다.

(요즘 회사 컴퓨터로 코딩을 하고 있으면, 부업 하냐고 핀잔을 듣습니다ㅜ)

잠시 후 회신이 왔습니다.

다행히 CopyPage, PastePage를 사용한 코드는 의도한 대로 잘 작동했다고 합니다.

아래는 insert_file 메서드 대신 CopyPage와 PastePage 액션을 대신 사용한 파이썬 코드입니다.

문서병합은 불가피하게 두 개 이상의 창을 열어야 하는데,

두 개의 프로세스를 사용하는 방식과, 하나의 프로세스에 두 개의 문서를 동시에 여는 방식이 있습니다.

두 가지 다 보여드리니, 업무자동화에 참고하시기 바랍니다.

① Hwp 인스턴스를 두 개 만들어서 병합하는 방법

개인적으로는 이 방식을 좋아합니다.

비록 메모리도 더 많이 먹고, 속도도 다소 느린 반면

코드가 비교적 직관적입니다.

import os
from pyhwpx import Hwp

hwp_list = [i for i in os.listdir() 
            if i.endswith(".hwp")]

hwp = Hwp()  # 기존 열린 한/글에 연결하거나, 새 창(hwp) 열기
hwp.open(hwp_list[0])  # 첫 번째 문서 열기
hwp.MoveDocEnd()  # 문서 마지막 페이지로 이동

hwp2 = Hwp(new=True, visible=False)  # 새 창(hwp2)을 다른 프로세스로 백그라운드에 열기
for h in hwp_list[1:]:  # 두 번째 파일부터 마지막 파일까지
    hwp2.open(h)  # 하나씩 hwp2에 열어서
    hwp2.MoveDocBegin()  # hwp2 문서 첫페이지로 이동
    for p in range(1, hwp2.PageCount + 1):  # 1페이지부터 마지막 페이지까지
        hwp2.goto_page(p)  # 해당 페이지로 이동해서(goto_page는 1페이지의 인덱스가 0이 아니라 1임)
        hwp2.CopyPage()  # hwp2의 현재쪽을 복사해서
        hwp.PastePage()  # hwp에 페이지 붙여넣기(커서가 중간에 있더라도 페이지단위로 잘 추가됨)
    hwp2.Close()  # 페이지를 모두 옮겼으면 해당파일 닫고 (다음파일 열 준비)
hwp2.Quit()  # 모든 문서의 병합을 마쳤으면 hwp2만 종료(hwp는 살아있음)

코드 이해를 돕기 위해 주석을 덕지덕지 달았는데,
오히려 지저분해 보이기도 하네요ㅜ
과정이 이해가 된다면, 코드를 보지 말고
편집기에서 코드를 완성해보세요.

② 새 창(또는 탭)을 추가해서 병합하는 방법

방식은 달라도 결과는 완전히 똑같습니다.

먼저 hwp 인스턴스를 생성한 후, hwp.add_doc()이나, hwp.add_tab()을 실행해줍니다.

add_doc은 새 창을 추가, add_tab은 새 탭을 추가하는 명령어입니다.

그 상태에서 두 번째 창에 2~4번 문서를 하나씩 열어서 첫 번째 창에 반복해서 붙여넣는 방식입니다.

창간 전환을 위하여 계속해서 hwp.switch_to()를 실행해줘야 하지만,

어차피 for문이나 while문 안에서 쓰이게 되므로 코드가 크게 복잡해지거나 하지는 않습니다.

import os
from pyhwpx import Hwp

hwp_list = [i for i in os.listdir() 
            if i.endswith(".hwp")]

hwp = Hwp()
hwp.open(hwp_list[0])  # 목록의 첫 번째 문서를 열어서
hwp.MoveDocEnd()  # 문서 끝으로 이동해놓고, (붙여넣을 준비)

hwp.add_doc()  # 빈 창 추가. (한 프로세스에서 여러 문서 동시조작 가능)

for h in hwp_list[1:]:  # 두 번째 문서부터 마지막 문서까지 하나씩
    hwp.open(h)  # 빈 창에다 열어서
    hwp.MoveDocBegin()  # 문서 처음으로 이동한 후
    for p in range(1, hwp.PageCount + 1):  # 1페이지부터 끝 페이지까지
        hwp.goto_page(p)  # 한 쪽씩 이동하면서
        hwp.CopyPage()  # 쪽복사
        hwp.switch_to(0)  # hwp[0]으로 돌아와서
        hwp.PastePage()  # 쪽붙이기
        hwp.switch_to(1)  # 붙여넣었으면 hwp[1]로 되돌아가기
    hwp.Close()  # 모든 페이지를 복사했으면 hwp[1]의 현재문서 닫기
hwp.FileClose()  # 두 번째부터 마지막 문서까지 복사를 마쳤으면 hwp[1] 창 닫기

여기까지 새 창을 이용한 방식이었습니다.

아래 보여드리는 마지막 코드는 새 탭을 이용하는 방식입니다.

add_doc을 add_tab으로, FileClose를 TabClose로 변경한 것 빼고는 완전히 똑같은 코드입니다.

import os
from pyhwpx import Hwp

hwp_list = [i for i in os.listdir() 
            if i.endswith(".hwp")]

hwp = Hwp()
hwp.open(hwp_list[0])
hwp.MoveDocEnd()

hwp.add_tab()  # add_doc으로 바꾸면 빈 창이 추가됨

for h in hwp_list[1:]:
    hwp.open(h)
    hwp.MoveDocBegin()
    for p in range(1, hwp.PageCount + 1):
        hwp.goto_page(p)
        hwp.CopyPage()
        hwp.switch_to(0)
        hwp.PastePage()
        hwp.switch_to(1)
    hwp.Close()
hwp.TabClose()

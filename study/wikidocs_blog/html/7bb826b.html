    ② 웹 개발, 데이터 분석, AI/ML 등 다양한 분야에 사용되는 범용 언어이다.
    ③ 방대한 라이브러리와 커뮤니티 지원이 특징이다.
    ④ 컴파일 언어로 실행 속도가 C언어보다 빠르다.

  2. 파이썬 설치 시 "Add python.exe to PATH" 옵션을 체크하는 주된 이유는 무엇인가?
    ① 파이썬의 실행 속도를 높이기 위해
    ② 터미널이나 명령 프롬프트 어디서든 파이썬을 실행할 수 있도록 하기 위해
    ③ 가상 환경을 자동으로 생성하기 위해
    ④ VS Code를 자동으로 설치하기 위해

  3. 윈도우 터미널(cmd)에서 설치된 파이썬의 버전을 확인하는 명령어로 옳은 것은?
    ① python --version
    ② python --check
    ③ python -v
    ④ check -python

  4. 파이썬 코드를 실행하는 방법 중 VS Code에서 새 파일을 작성한 후 사용하는 단축키는?
    ① Ctrl + C
    ② Shift + F5
    ③ Alt + Tab
    ④ Ctrl + Z

  5. 파이썬에서 프로젝트별로 독립적인 환경을 구축하여 라이브러리 버전 충돌을 방지하는 기능을 무엇이라 하는가?
    ① 통합 개발 환경(IDE)
    ② 가상 환경(Virtual Environment)
    ③ 클라우드 환경
    ④ 라이브러리 관리자

  6. 일반적인 파이썬 스크립트 파일의 확장자로 옳은 것은?
    ① .txt
    ② .ipynb
    ③ .py
    ④ .java

  7. 다음 중 주피터 노트북(Jupyter Notebook) 파일 형식에 대한 설명으로 옳은 것은?
    ① 확장자는 .py이다.
    ② 코드, 실행 결과, 문서(Markdown)를 함께 관리할 수 있다.
    ③ 오직 텍스트 데이터만 저장할 수 있다.
    ④ 실제 배포용 대규모 개발 프로젝트에 가장 적합한 파일 형식이다.

  8. 주피터 노트북 파일의 확장자는 무엇인가?
    ① .py
    ② .html
    ③ .ipynb
    ④ .doc

  9. 파이썬의 특징 중 하나로, 특히 어떤 분야에서 가장 많은 활용성을 보이는가?
    ① 운영체제 개발
    ② LLM(대규모 언어 모델) 및 AI 개발
    ③ 하드웨어 제어
    ④ 모바일 앱 전용 개발

  10. 터미널에서 파이썬 대화형 모드(Interpreter)를 종료하지 않고 코드를 한 줄씩 실행할 때 나타나는 프롬프트 기호는?
    ① >>>
    ② ###
    ③ :::
    ④ ---

  11. AI와 머신러닝 개발에 파이썬이 가장 많이 쓰이는 이유와 거리가 먼 것은?
    ① 간결한 문법
    ② 강력한 데이터 처리 라이브러리 존재
    ③ 낮은 메모리 점유율과 하드웨어 직접 제어 능력
    ④ AI 커뮤니티의 활발한 지원

  12. 파이썬 개발 환경 구축 시 주로 권장되는 에디터는?

    ② Visual Studio Code
    ③ 그림판

    ② 프로젝트마다 필요한 라이브러리 버전이 다를 때 발생하는 충돌을 방지하기 위해
    ③ 보안을 강화하기 위해
    ④ 유료 라이브러리를 무료로 쓰기 위해

  14. VS Code에서 파이썬 코드를 작성한 후, 터미널에서 실행하기 위한 명령어 형식은?

    ④ start [파일명].py

  15. 주피터 노트북(.ipynb) 환경의 특징에 대한 설명으로 틀린 것은?
    ① 코드와 실행 결과, 마크다운 문서를 한데 묶어 관리할 수 있다.
    ② 셀(Cell) 단위로 코드를 실행할 수 있어 데이터 분석에 유리하다.
    ③ 파일 내부적으로는 JSON 형식으로 저장된다.
    ④ 대규모 웹 서비스 배포용 소스 코드를 작성할 때 주로 사용된다.

  16. 다음 중 파이썬의 '예약어(Keyword)'가 아닌 것은?

  17. 파이썬에서 문자열 안에 변수 값을 쉽게 넣기 위해 사용하는 'f-string'의 올바른 표기법은?

    ④ append() 함수를 사용해 요소를 추가할 수 있다.

  19. 반복문에서 현재 루프를 중단하고 즉시 반복문을 빠져나갈 때 사용하는 키워드는?

Mac에서 중복된 MP4 파일을 삭제하는 방법 - 완벽한 가이드

@philmartincollins
2026년 3월 19일 9:25 오후
중복 Mp4 파일

개요 - Mac에서 중복된 MP4 동영상 몇 개만 찾아 삭제하는 방법을 알아본 적이 있다면, 이 글이 바로 여러분이 찾던 내용임을 알게 될 것입니다. 다음은 “Mac에서 중복된 MP4 파일을 어떻게 삭제하나요?”라는 질문을 상세히 다룬 단계별 가이드입니다.

현재의 디지털 과잉 시대에 동영상 파일은 이미 우리 엔터테인먼트 생활에서 가장 흔하거나 필수적인 요소 중 하나가 되었습니다. 그래서 우리는 소중한 추억을 담을 폴더를 하나 만들지만, 어느새 그 폴더는 통제 불능 상태가 되어버리곤 합니다. 마치 아이들이 의자 놀이를 하다 보면 결국 모든 참가자가 한 자리에 모이게 되는 것과 같습니다. 이 과정에서 무심코 중복 파일을 다운로드하게 되고, 이는 파일 구조를 혼란스럽게 만들어 결국 대혼란을 초래합니다.

게다가 특정 동영상 파일을 찾으려 할 때면 일이 더 어려워질 수 있습니다. 따라서 이러한 문제를 해결하기 위해, MP4 동영상을 한 번에 정리할 수 있는 수동 및 자동 방법을 추천해 드립니다. 때로는 JPEG, PNG, GIF 등 다른 파일 형식을 마주칠 수도 있습니다. 이러한 상황에 대비해, 저희는 다음과 같은 가이드를 제공합니다 delete duplicate photos on PC?”

Mac에 중복 MP4 파일을 보관할 때 고려해야 할 사항

하지만 Mac OS에서 시간과 비용을 절약하고 싶다면:

중복 파일은 Mac의 소중한 디스크 공간을 차지합니다.
중복 파일을 제거하는 별도의 지침이 반드시 필요합니다.
중복 파일로 인해 백업 용량이 필요 이상으로 커집니다.
중복 파일이 있으면 원본 MP4 파일이 어느 것인지 구분할 수 없습니다. 시간이 지날수록 문제는 악화되어 결국 모든 파일이 손상되거나 사라질 수 있습니다!
시스템에 장애가 발생해 복구되지 않을 경우, MP4 파일이 여전히 정상인지 어떻게 확신할 수 있겠습니까?
중복 파일이 있으면 MP4 파일 검색 속도가 훨씬 느려집니다.
수동 작업 시 오류가 발생하거나 MP4 파일이 사라질 수 있습니다.

Mac OS에서 파인더를 통해 중복 MP4 파일을 삭제하는 방법은?

중복 파일을 remove duplicate files: 파인더 앱 사용

파인더에서 앱 디렉터리를 탐색하다 보면 ‘MP4 파일’이라는 이름의 폴더가 보일 것입니다(앞서 설명한 대로 H.264 MP4 변환 소프트웨어를 설치했다면, 이 폴더는 ‘응용 프로그램’ 폴더에서 찾을 수 있습니다).

동영상 저장 폴더에서 ‘목록’ 보기로 전환한 후 이름이나 크기 순으로 정렬하세요.

각 파일을 하나씩 확인하며, 페이지마다 반복되는 동일한 내용의 복사본을 그냥 넘기지 말고 같은 날짜의 MP4 파일을 선택하세요. (어떤 파일 하나를 예시로 들어보겠습니다.)

이제 Command 키를 누른 상태에서 중복 파일을 마우스 오른쪽 버튼으로 클릭하고 항목 메뉴에서 “휴지통으로 이동”을 선택하세요.

터미널을 통해 macOS에서 중복 멀티미디어 파일을 찾고 제거하는 방법은 무엇인가요?

미치기 전에 정신 차리기: 미디어 라이브러리에서 동일한 내용을 가진 MP4 파일을 미친 듯이 찾아야 한다면, 여기 기술적인 단계가 나와 있습니다.

아래 단계를 실행하려면 터미널 명령어에 대한 기본적인 실무 지식이 필요합니다.

이 터미널을 사용하세요 - ‘응용 프로그램’ → ‘유틸리티’ → ‘터미널’로 이동합니다.

비디오 폴더로 이동 - ‘cd’ 명령어를 사용하여 비디오 폴더 안으로 들어갑니다(예: open-terminal-cd).

중복 파일 찾기 - 다음 명령어를 입력하세요:

find. -type f -print0 | xargs -0 md5sum |sort | uniq -w 32 -d | cut -c 33 -

중복 파일 제거 - 직접 수행하세요: ‘rm’ 명령어를 사용하여 중복 파일을 찾아 이동합니다(예: rm / 중복 / 비디오 / 경로).

주의 사항: 파일에 손상을 입히거나(혹은, 끔찍하게도 되돌릴 수 없는) 변경을 가하기 전에는 항상 파일을 백업하십시오.

터미널 명령어로 삭제된 파일은 복구할 수 없습니다.

Mac의 ‘사진’ 앱으로 중복 MP4 파일을 삭제하는 방법은?

사용자들은 이 기능이 그다지 매력적이지 않다고 생각하는 편입니다. ‘사진’ 앱은 주로 사진을 관리하기 위한 것이지만, 사실 동영상 관리에도 더 편리합니다. 물론 동영상 관리에도 충분히 활용할 수 있습니다!

그럼, 어떻게 해야 할까요?

먼저, ‘사진’ 앱을 실행해 봅시다.

Mac에서 ‘사진’ 앱을 엽니다.

MP4 파일을 하드 드라이브로 가져오기

MP4 파일을 넣으려면 드래그하기만 하면 됩니다. 아직 ‘사진’ 라이브러리에 없는 MP4 파일의 경우 '가져오기'를 사용하세요.

스마트 앨범 만들기

메뉴 막대에서 '파일'을 클릭하세요. 드롭다운 메뉴에서 '새 스마트 앨범...'을 선택하세요.

파일 이름이나 날짜와 같은 속성 기준 설정

그러면 스마트 앨범이 해당 조건에 맞는 MP4 파일을 필터링하여 중복 파일이 있는 위치를 보여줍니다.

스마트 앨범 탐색하기

스마트 앨범에서 이름이 동일한 MP4 파일의 중복본을 검색하세요.

중복 MP4 파일 삭제하기

스마트 앨범 내에서 중복 파일을 선택하고 “삭제”를 누르세요. 그런 다음 확인을 누르세요.

최근 삭제된 항목 제거

삭제된 MP4 파일은 “최근 삭제된 항목”으로 이동합니다. 해당 폴더로 이동하여 모든 항목을 선택한 후 “모든 항목 제거” 또는 “복원”을 클릭하세요.

중복 MP4 파일을 제거하는 무료 솔루션의 단점

중복 MP4 파일을 정리하려는 사용자가 직면하는 몇 가지 문제점은 다음과 같습니다.

일부 방법은 파일을 실제로 삭제해야 하는 반면, 다른 방법은 컴퓨터 성능을 다양한 수준으로 저하시키기도 합니다.
무료 도구를 사용할 경우 실수로 파일을 삭제하여 데이터 손실이 발생하기 쉽습니다.
수동 삭제 시 최적의 저장 공간을 확보하지 못할 수도 있습니다.
대용량이고 지속적으로 증가하는 데이터 세트의 경우 이러한 단순한 방법을 사용하는 것은 불가능합니다.
파일 명명 규칙은 정확성을 제한하지만 여전히 널리 사용되고 있습니다.
무료 중복 제거 도구는 정기적인 조정이 필요합니다.
수동 프로세스의 위험 요소로는 보안 취약점이 포함됩니다.
따라서 정상 버전과 중복 파일을 구별하기가 매우 어렵습니다.
자동화 없이는 규정 준수가 어렵습니다.
현재의 방법들은 중복 제거에 있어 까다로운 요구 사항을 가지고 있습니다.

macOS에서 중복 MP4 파일을 제거하는 방법 - 가장 이상적인 방법

현재 몇 가지 단점이 있지만, SysTools Duplicate File Finder 소프트웨어를 컴퓨터의 시스템 디렉터리에 설치하는 것입니다. 이를 통해 모든 것이 완전히 정리된 후, 외부 요소의 간섭 없이 적절히 선택된 시간대에 자체 시스템을 실행할 수 있게 됩니다.

이 도구의 장점을 활용하면, 몇 번의 마우스 클릭만으로 중복 MP4 파일을 제거할 수 있습니다. 지능형 파일 스캔 엔진: 빠른 스캔
재귀적 스캔 옵션
중복 동영상 파일 이동 또는 삭제
70가지 이상의 파일 형식 스캔 (예: mp3, mp4, png, jpg)
시각적으로 동일한 이미지 감지
스캔 후 사용자 데이터는 그대로 유지됩니다.
지금 무료 버전을 사용해 보세요!

Mac에서 중복 .mp4 파일을 스캔하고 제거하는 중요한 단계는 다음과 같습니다:

중복 동영상 파일 찾기 도구를 다운로드하여 설치하세요.

“폴더 추가”를 클릭하여 컴퓨터에서 파일이 있는 위치를 선택하세요.

대화 상자에서 도구가 나중에 처리할 스캔 대상(파일 형식: MP4, AVI 등)에 대한 입력 매개변수를 설정해야 합니다. “계속”을 클릭하세요.

결론

이 내용을 통해 macOS Ventura에서 MP4 파일을 삭제하는 방법에 대해 대략적인 이해를 하셨을 것입니다. macOS Ventura 및 이전 버전을 사용하는 시스템에서 중복된 동영상 클립을 제거하는 데는 수동 방식과 전문적인 솔루션이 있습니다. 수동 방식은 무료이지만 전문적인 기술이 필요합니다. 바로 이럴 때 전용 프로그램이 유용합니다.

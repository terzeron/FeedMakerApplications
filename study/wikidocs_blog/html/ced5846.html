@dragonhappy9
2025년 1월 21일 2:11 오후
Springboot

여기에 간편하게 프로젝트를 만들수 있도록 돕는 사이트가 있습니다.
https://start.spring.io/

여기서 보셔야 하는것들에 대한 설명입니다.
Project : 프로젝트의 빌드 자동화 툴
Language : 프로젝트에서 사용할 언어
Spring Boot : Spring Boot의 버전
Project Metadata : 프로젝트에 대한 Metadata 즉 프로젝트에 대한 데이터설정
Group - 회사명
Artifact & Name - 프로젝트 이름
Description - 프로젝트에 대한 설명
Package name - 프로젝트의 기본 패키지 이름
Packaging : 프로젝트를 패키징할 수단
JAVA : JAVA 언어를 통해 개발할 때 기본적으로 제공받는 라이브러리 버전
Dependencies : 프로젝트를 생성하고 Property에서 라이브러리를 사용하기위해 필요한 의존성을 주입을 하는것과 같습니다.
저는 Spring Web, JPA, Lombok, Security, Dev Tools, MySQL Driver를 주입햇습니다.
Spring Web - 프레임워크
JPA - 데이터베이스를 간편하게 사용할수 있도록 도와주는 라이브러리
Lombok - 어노테이션을 제공하고, 이것을 통해 코드를 자동으로 생성해주는 라이브러리 ex) @Getter -> Getter 자동생성
Spring Security - 사용자 인증, 접근보호, 등등을 지원하는 보안 라이브러리
DevTools - 개발자가 유용하게 사용할수 있는 도구(서버를 재시작할필요 없이 변경된 코드를 적용하는 등의 기능)
MySQL Driver - Mysql에 연결하기 위한 드라이버
Thymeleaf - 웹 페이지를 서버에서 데이터를 넣어 렌더링할수 있도록 돕는 라이브러리

이후에 Generate를 누르면 압축파일을 다운받게 되는데, 이파일을 프로젝트 폴더에 넣으면 됩니다.

Android 15 + Magisk + Burp Suite 프록시 설정

Android 15 + Magisk + Burp Suite 프록시 설정 가이드

1. 환경 및 선수 조건

필요 도구

도구 설명
Magisk 29.0 Android 루팅 및 모듈 관리
Burp Suite 프록시 및 인증서 발급
Python HTTP 서버로 인증서 전송
OpenSSL 인증서 형식 변환 (필요 시)

네트워크 구성

  • Android 기기와 PC가 동일한 Wi-Fi 네트워크에 연결되어 있어야 함

2. AlwaysTrustUserCerts 모듈 설치

모듈 다운로드

https://github.com/NVISOsecurity/MagiskTrustUserCerts/releases/download/v0.4.1/AlwaysTrustUserCerts.zip

Magisk에서 설치

  1. 다운로드한 AlwaysTrustUserCerts.zip을 기기로 전송
  2. Magisk 실행 → 하단 모듈 탭 클릭
  3. 저장소에서 설치 → AlwaysTrustUserCerts.zip 선택
  4. 설치 완료 후 다시 시작 버튼으로 재부팅

3. Burp 인증서 발급

CA Certificate 다운로드

  1. Android 기기에서 Burp 프록시가 설정된 상태로 브라우저 열기
  2. http://burp 접속
  3. CA Certificate 클릭하여 cacert.der 다운로드

파일 형식 이해

인증서는 데이터 형식과 확장자가 분리된 개념

형식 설명
DER 바이너리 형식 (사람이 읽을 수 없음)
PEM Base64 텍스트 형식 (-----BEGIN CERTIFICATE----- 로 시작)
확장자 주로 담긴 내용 주로 쓰는 곳
.der DER (바이너리) Java, Android
.cer DER 또는 PEM Windows
.crt PEM 또는 DER Linux, Android

.cer와 .crt는 사실상 동일하며 플랫폼 관습 차이일 뿐

4. Python HTTP 서버로 인증서 전송

PC에서 HTTP 서버 실행

# 인증서가 있는 디렉토리로 이동
cd 인증서_경로

# HTTP 서버 실행 (기본 포트 8000)
python -m http.server 8000

Android에서 다운로드

  1. PC의 IP 확인 (예: 192.168.0.10)
  2. Android 브라우저에서 http://192.168.0.10:8000 접속
  3. cacert.der 파일 다운로드

5. 인증서 형식 변환 (DER → CRT)

1차 시도: 확장자만 변경

cacert.der  →  cacert.crt

파일 탐색기에서 확장자만 .crt로 변경 후 설치 시도

2차 시도: OpenSSL로 PEM 변환 (1차 실패 시)

openssl x509 -inform DER -in cacert.der -out cacert.crt

이 명령어는 DER 바이너리를 실제 PEM 형식으로 변환

6. Android에 인증서 설치

설치 경로

설정 > 생체인식 및 보안 > 기타 보안 설정 > 디바이스에 저장된 인증서 설치 > CA 인증서

⚠️ VPN 및 앱 사용 이 아닌 반드시 CA 인증서 선택

설치 순서

  1. 변환한 cacert.crt 선택
  2. 인증서 이름 설정 후 확인
  3. 설정 > 생체인식 및 보안 > 기타 보안 설정 > 인증서 관리 > 사용자 탭에서 설치 확인

시스템 인증서로 자동 승격 확인

  1. 기기 재부팅
  2. AlwaysTrustUserCerts 모듈이 사용자 인증서를 시스템 인증서로 자동 승격
  3. 인증서 관리 > 시스템 탭에서 Burp 인증서 확인

7. Burp 프록시 리스너 설정

  1. Burp Suite 실행
  2. Proxy > Proxy Settings > Proxy Listeners 에서 Add 클릭
  3. 설정:
  4. Bind to port: 원하는 포트 (예: 8080)
  5. Bind to address: All interfaces 선택 (모든 대역 수신)
  6. OK 후 리스너 활성화 확인

8. Android Wi-Fi 프록시 설정

  1. 설정 > 연결 > Wi-Fi 에서 연결된 네트워크 선택
  2. 고급 메뉴 → 프록시 → 수동 선택
  3. 설정 입력:
  4. 프록시 호스트 이름: PC의 IP 주소 (예: 192.168.0.10)
  5. 프록시 포트: Burp 리스너 포트 (예: 8080)
  6. 저장 후 브라우저 또는 앱 실행 → Burp에서 트래픽 캡처 확인

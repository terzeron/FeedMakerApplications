Python 딕셔너리에서 items()로 정렬 시 키 또는 값으로 정렬하는 방법

@SeoulCodeData
2025년 1월 8일 2:08 오후
자료구조 파이썬 Python

딕셔너리의 키와 값을 한쌍으로 묶어서 정렬해야하는 경우가 생깁니다. 이 때는 딕셔너리명.items()를 사용하면 됩니다.

mart = {'snack': 1500, 'icecream': 1800, 'notebook': 4000, 'meat': 20500, 'milk': 4500}
sorted(mart.items())
"""
[('icecream', 1800), ('meat', 20500), ('milk', 4500), ('notebook', 4000), ('snack', 1500)]
이 때는 키:값 쌍을 모두 보여주되, 정렬은 키 기준
"""

값을 기준으로 정렬하려면 operator라는 모듈이 필요합니다.

operator에는 계산을 도와주는 다양한 명령어가 포함되어 있습니다.

operator 중 itemgetter() 메서드를 사용해봅시다.

itemgetter()는 기준에 따라 키와 값을 모으는 명령어 입니다.

import operator

mart = {'snack': 1500, 'icecream': 1800, 'notebook': 4000, 'meat': 20500, 'milk': 4500}
print(sorted(mart.items(), key=operator.itemgetter(1)))
"""
>>> print(sorted(mart.items(), key=operator.itemgetter(1)))
[('snack', 1500), ('icecream', 1800), ('notebook', 4000), ('milk', 4500), ('meat', 20500)]
sorted 메서드에 추가된 key는 정렬 기준을 나타내는 옵션입니다.
operator.itemgetter()의 인자로 들어가는 값이 0이면, 키를 기준으로, 1이면 값을 기준으로 정렬합니다. 

이경우에는 값을 기준으로 정렬하였습니다.
"""

sorted()는 딕셔너리를 정렬해서 보여줄 뿐 딕셔너리에 저장된 내용에는 영향을 주지 않습니다. 정렬한 결과는 원래 딕셔너리와는 완전히 별개입니다.

따라서 정렬한 결과를 기억해야 한다면 변수에 저장해야 합니다.


mart = {'snack': 1500, 'icecream': 1800, 'notebook': 4000, 'meat': 20500, 'milk': 4500}
mart = sorted(mart.keys())
"""
>>> mart

['icecream', 'meat', 'milk', 'notebook', 'snack']
정렬된 결과를 딕셔너리로 저장하려면 dict()로 감싸 데이터타입을 딕셔너리로 변환하면 됩니다. 
하지만 딕셔너리는 키와 값이 동시에 필요하므로 items()로 정렬한 결과만 딕셔너리로 변환할 수 있습니다.

>>> mart = {'snack': 1500, 'icecream': 1800, 'notebook': 4000, 'meat': 20500, 'milk': 4500}
>>> mart = dict(sorted(mart.items()))
>>> mart
{'icecream': 1800, 'meat': 20500, 'milk': 4500, 'notebook': 4000, 'snack': 1500}
"""

2-2. 딥리서치 활용법 실습

@myungsang
2025년 11월 6일 10:30 오전
수어통역센터강의자료

📋 실습 개요

목표: Gemini Deep Research로 전문 리서치 보고서 자동 생성

완성물: 선택한 주제의 심층 분석 보고서 (10-15페이지)

시간: 소개(5분) + 실습1(15분) + 실습2(15분) + 정리(5분)

🎯 딥리서치란?

일반 AI vs 딥리서치 차이:

  • 일반 AI: 1번 검색 → 즉시 답변 (1-2분)
  • 딥리서치: 여러 사이트 반복 검색 → 종합 분석 (5-10분)

언제 사용?

  • 정책 자료 수집
  • 사례 조사
  • 통계 분석
  • 보고서 작성

필요조건:

  • 무료로 가능하나 Gemini Advanced 유료 구독 권장 (월 3만원)

📝 실습 주제 4가지

주제 1: 청각장애인 고용 정책 2024-2025

  • 정부 지원금 변화
  • 의무고용률 현황
  • 우수 고용 기업 사례

주제 2: AI 수어통역 기술 발전

  • 최신 AI 수어인식 기술
  • 국내외 개발 현황
  • 상용화 가능성

주제 3: 전국 수어통역센터 우수 사례

  • 지역별 운영 모델
  • 혁신 서비스 사례
  • 벤치마킹 포인트

주제 4: 세종시 청각장애인 현황 분석

  • 등록 장애인 수 및 비율
  • 복지 서비스 이용 현황
  • 개선 필요 사항

실습 진행: 수강생 자율 주제 선택

🔍 실습 1: 딥리서치 시작하기 (15분)

1단계: 딥리서치 실행

Gemini Advanced 접속:

  1. https://gemini.google.com
  2. 왼쪽 사이드바 → 'Deep research' 클릭
  3. 새 리서치 시작

2단계: 프롬프트 입력

예시 - 주제 1 선택 시:

"2024-2025년 청각장애인 고용 정책 분석

조사 내용:
1. 정부 지원금 제도 변화 (2023년 대비)
2. 장애인 의무고용률 현황 및 미달 기업 제재
3. 청각장애인 고용 우수 기업 3곳 사례
4. 세종시 청각장애인 취업률 통계
5. 2025년 신규 정책 예고

출처:
- 정부 공식 사이트 우선
- 최신 뉴스 (2024년 이후)
- 통계청, 고용노동부 자료

형식:
- 각 항목별 요약
- 표와 그래프 포함
- 출처 링크 명시"

3단계: 리서치 진행 관찰

화면 표시 내용:

  • "웹에서 정보 수집 중..." (30초-1분)
  • "○○ 사이트 분석 중" (검색 과정 실시간 표시)
  • "보고서 작성 중..." (2-3분)

대기 시간 활용:

  • 다른 작업 가능 (이메일, 문서 작성)
  • 완료되면 알림

4단계: 결과 확인 및 활용

생성된 보고서:

  • 10-15페이지 분량
  • 목차 자동 구성
  • 출처 링크 40-50개
  • 표/그래프 포함

다운로드:

  • '내보내기' → Google Docs
  • 또는 텍스트 복사 → 한글/워드

🔍 실습 2: 추가 질문 및 심화 (15분)

추가 질문 예시

생성된 보고서 아래 추가 프롬프트:

"위 내용을 바탕으로:

1. 세종시 수어통역센터에서 참고할 만한 
   고용 연계 프로그램 3가지 제안해줘

2. 타 지역 센터 대비 세종시 장점/단점 분석

3. 내년 사업계획서에 넣을 정책 근거 정리
   (1페이지 분량, 공문서 형식)"

결과:

  • 맞춤형 제안서 생성
  • 즉시 업무 활용 가능
  • 추가 검색 없이 심화 분석

💡 활용 시나리오

센터 업무 활용

1. 월간 보고서 작성

  • 주제: "청각장애인 복지 정책 월간 동향"
  • 시간: 3시간 → 15분 단축

2. 사업계획서 근거 자료

  • 주제: "신규 사업 필요성 및 타당성"
  • 통계 + 사례 자동 수집

3. 벤치마킹 자료

  • 주제: "타 지역 센터 혁신 사례"
  • 5-10곳 사례 한 번에 정리

4. 교육 자료 제작

  • 주제: "최신 수어 교육 트렌드"
  • 커리큘럼 개발 참고

⚠️ 주의사항

정확도 확인:

  • 출처 링크 직접 클릭하여 검증
  • 중요 통계는 원본 사이트 확인
  • 최종 보고서 제출 전 팩트체크 필수

비용:

  • Gemini Advanced 월 구독 필요
  • 무료 사용자는 일반 검색 활용
  • 센터 공용 계정 고려

시간:

  • 복잡한 주제: 5-10분 소요
  • 대기 중 다른 작업 병행 권장
  • 실패 시 프롬프트 단순화 후 재시도

🆚 일반 검색 vs 딥리서치 비교

구분 일반 검색 딥리서치
시간 30-60분 5-10분
출처 수 3-5개 40-50개
분석 깊이 얕음 깊음
자동 정리 수동 자동
비용 무료 유료

추천:

  • 간단한 정보: 일반 검색
  • 전문 리서치: 딥리서치
  • 보고서 작성: 딥리서치

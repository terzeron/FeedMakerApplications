터미널에서 구글 워크스페이스를 정복하다: 새로운 Google Workspace CLI (gws) 소개

@JNCD
2026년 3월 5일 8:05 오전

구글 워크스페이스(Google Workspace)를 사용하는 개발자나 자동화에 관심 있는 분들이라면 눈여겨봐야 할 아주 따끈따끈한 도구를 소개해 드리고자 합니다.
바로 Google에서 최근 공개한 Google Workspace CLI (코드명: gws)입니다.
1. Google Workspace CLI (gws)란?
기존에 Google Apps Script를 로컬에서 관리하기 위해 clasp를 사용했다면, 이 새로운 CLI는 그 범위를 훨씬 넓혔습니다. Drive, Gmail, Calendar, Sheets, Docs, Chat, Admin 등 Google Workspace의 거의 모든 API를 단 하나의 명령줄 도구로 제어할 수 있게 해줍니다.
가장 놀라운 점은 구글의 Discovery Service를 통해 명령어를 동적으로 생성한다는 것입니다. 즉, 구글이 새로운 API 기능을 추가하면 이 CLI도 자동으로 그 기능을 인식하게 됩니다.
2. 주요 특징 (Why gws?)
① 사람과 AI 모두를 위한 설계
* 인간 개발자에게: 복잡한 curl 호출이나 REST 문서를 뒤질 필요가 없습니다. 탭 자동 완성, --help 지원, 실행 전 미리 보기를 위한 --dry-run 기능을 제공합니다.
* AI 에이전트에게: 모든 응답이 정형화된 JSON으로 출력됩니다. LLM(대형 언어 모델)이 워크스페이스 데이터를 읽고 처리하기에 최적화되어 있습니다.
② 40개 이상의 AI 에이전트 스킬 포함
단순한 API 호출을 넘어, 이 저장소에는 AI가 이메일을 분류하거나 캘린더 일정을 관리하는 등의 복잡한 작업을 수행할 수 있도록 돕는 'Agent Skills'가 포함되어 있습니다.
③ 강력한 보안 및 인증
* 로컬 데스크톱에서는 OS 키체인을 활용한 암호화 저장을 지원합니다.
* CI/CD 환경이나 서버를 위해 서비스 계정(Service Account) 및 액세스 토큰 방식도 완벽하게 지원합니다.
3. 간단 사용법 맛보기
설치하기 (Node.js 기반)
npm install -g @googleworkspace/cli

초기 설정
gws setup

(이 명령어를 통해 구글 클라우드 프로젝트 설정과 OAuth 로그인을 한 번에 진행할 수 있습니다.)
실전 명령어 예시
* 드라이브 파일 목록 보기:
gws drive files list --params '{"pageSize": 5}'

  • 새로운 스프레드시트 생성:
    gws sheets spreadsheets create --json '{"properties": {"title": "2024 예산안"}}'

  • 파일 업로드:
    gws drive files create --json '{"name": "보고서.pdf"}' --upload ./report.pdf

  • 왜 이 도구가 중요한가요?
    지금까지 Google Workspace API를 사용하려면 각 언어별 클라이언트 라이브러리를 설치하고 인증 로직을 짜는 번거로운 과정이 필요했습니다. 하지만 gws를 사용하면:

  • 셸 스크립트만으로도 강력한 구글 워크스페이스 자동화 도구를 만들 수 있습니다.
  • Rust로 작성되어 매우 빠르고 가볍습니다.
  • JSON 출력 덕분에 jq 같은 도구와 조합하여 데이터를 가공하기 매우 쉽습니다.
    마치며
    현재 이 프로젝트는 활발하게 개발 중인 단계(v1.0 미만)이므로 일부 변경 사항이 생길 수 있지만, 구글 워크스페이스를 업무의 중심에 두고 계신 분들에게는 혁신적인 도구가 될 것입니다.

데이터 자동화 관리

@jae904
2025년 9월 8일 1:44 오후
  1. 필요성
    반복적으로 수행되는 업무 중에는 데이터 기반으로 판단하고 처리해야 하는 작업들이 많습니다. 이러한 업무는 대개 업무 시간 중 1~2시간 이상을 소모하며, 수많은 엑셀 시트를 눈이 빠지도록 들여다보는 과정을 포함합니다.
    나름대로는 COUNTIF 같은 함수들을 적극 활용하며 업무 효율을 개선해왔다고 여겼지만, 막상 결과를 결재 단계에 올려놓고 보면 한두 개씩 실수가 발견되는 경우도 많고, 반복 작업으로 인한 매너리즘에 쉽게 빠지게 됩니다.
    과연, 지금의 방식이 정확성과 완성도를 담보할 수 있는가? 여전히 그 질문 앞에서는 뚜렷한 한계와 벽을 느끼게 됩니다.

‘빠르게 방대한 양의 데이터에서 원하는 정보를 수집하고 싶다.’
‘데이터 정보를 빠르게 처리하고 싶다.’
‘중복적인 행정 업무를 줄이고 싶다.’

이런 배경에서 제가 현재 사용 중인 것들을 ai를 통한 업무 자동화를 진행해보았습니다.

  1. 준비하기
  2. 챗 GPT(저는 유료버전이용 했습니다.) / 스프레드시트OR엑셀 / 앱스크립트 OR VBA 모듈

  3. 자동화를 도모하고 싶은 업무(명확한 방향성)

  4. 진행순서.
    -> 챗 GPT 프롬포트 입력
    “CMS / 통장 후원 내역을 정리 해놓고, 비지정 / 지정(각 사업별 이름으로 구분 해놓으면 자동으로 이름 / 금액 / 지정(사업명)OR비지정으로 구분될 수 있도록 후원 마감 시트를 만들고 싶어.”

-> 스크립트 / 모듈 형성

-> 액셀 매크로(VBA) OR 앱스크립트 복사 / 붙여넣기

*앱스크립트의 경우, 최초 실행 시 경고와 함께 권한 관련 경고메시지가 뜹니다. 이 때 하단 “고급 서비스”에 들어가셔서 약관 서비스 동의를 하시면 앱스크립트가 실행됩니다.

  • 이번 후원금 마감시트에 적용된 것은
    1번을 누르면 CMS에서 처리되는 후원자명과 기관에서 관리 중인 후원자 이름을 맞추는 것, 2번을 누르면 시트 2번에 넣어놓은 후원자별 지정./비지정 구분에 맞추어 분류 순으로 진행했습니다. 아래는 적용 내용을 캡쳐 떠놓은 것입니다

  • 도중도중 수식에 대한 오류들은 그 대화상자를 그대로 챗 GPT에 넣으면 관련된 내용을 스크립트를 다시 짜줍니다.

  • 적용하며 나름 꿀팁 몇가지를 소개하자면,
  • “전체적인 수식을 다시 작성해달라.”는 요청을 해서 수정된 전체 수식을 넣어야 식이 꼬이지 않습니다.
  • 시트나 원하는 데이트가 있는 셀을 정확하게 입력을 해야 “해당 시트를 찾을 수 없다.” 든가 “데이터를 찾을 수 없다.”는 오류가 발생하지 않습니다.
  • 매번 VBA나 앱스크립트에 들어가서 실행을 누르는 것이 번잡스럽기 때문에 도형이든 이미지든 삽입해 스크립트를 연결시켜주시는 것이 좋습니다.

정리하면 간단합니다.
챗 GPT 스크립트 OR 모듈 요청 -> 스크립트, 모듈 적용 -> 오류 수정 -> 수정 스크립트, 모듈적용 -> 이미지 스크립트 할당 -> 적용
*단, 오류를 잡고 적용하는 것이 복잡할수록, 시간이 오래걸립니다. 하지만 하루의 2~3시간 투입으로 업무수행기간동안 계속 사용할 수 있는 업무 툴을 만들 수 있는 것은 정말 큰 메리트라고 생각합니다!

저는 후원마감시트 자동화(내부), 후원자 관리 업데이트, 일지 기록 자동화를 진행했습니다.

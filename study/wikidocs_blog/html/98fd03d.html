인라인AI란 무엇인가?

안녕하세요! 매일 한글 문서와 씨름하며 사업계획서, 보고서 작성에 시간을 쏟고 계신가요? 오늘은 2025년 4월 정식 출시된 인라인AI에 대해 완벽하게 정리해드리겠습니다.

인라인AI는 한글(HWP) 문서 위에서 바로 작동하는 AI 비서입니다. 현재 전문가, 교육자, 공직자 23,000명이 매일 4시간씩 사용하고 있는 국내 대표 문서 작성 AI입니다.

인라인AI의 특별한 점

ChatGPT나 다른 AI 도구와 가장 큰 차이점은 무엇일까요?

바로 아래아한글 프로그램에 직접 연동되어 복사/붙여넣기 없이 문서를 완성한다는 점입니다.

인라인AI의 핵심 기능

1️⃣ 양식에 맞춘 자동 문서 작성

복잡한 표, 각주, 문단 구조까지 양식에 완벽하게 맞춰 내용을 자동으로 채워줍니다.

주요 특징:
- 문단, 표, 각주 등 복잡한 양식 자동 완성
- 사업계획서의 복잡한 표 자동 입력
- 별도 학습 없이 즉시 사용 가능

2️⃣ 다양한 파일 형식 인식

지원 파일 형식 설명
한글 (HWP) 기본 문서 형식
워드 (DOCX) MS 워드 문서
PDF PDF 문서 분석
이미지 이미지 텍스트 추출
엑셀 (XLSX) 데이터 분석
PPT 프레젠테이션 자료

과거 자료를 기반으로 새로운 문서를 만들 때 특히 유용합니다.

3️⃣ 로컬 RAG 기반 보안 시스템

가장 중요한 보안 기능입니다. 사용자의 자료를 외부로 전송하지 않고 로컬 환경에서만 분석합니다.

보안 장점:
- ✅ 민감한 법률 문서 안전 처리
- ✅ 기업 내부 자료 보안 유지
- ✅ 개인정보 보호 강화

ChatGPT vs 인라인AI, 뭐가 다를까?

많은 분들이 궁금해하시는 부분입니다. 비교표로 정리해드리겠습니다.

구분 ChatGPT 인라인AI
문서 연동 별도 복사/붙여넣기 필요 한글 프로그램 직접 연동
양식 처리 수동 조정 필요 자동으로 양식에 맞춰 배치
표 작성 텍스트로만 생성 표 안에 자동 입력
각주 삽입 수동 작업 자율적 각주 삽입
보안 클라우드 전송 로컬 환경 처리

한컴 AI / MS Copilot과의 차이

한컴 AI와 MS Copilot도 텍스트 생성은 가능하지만, 내용을 넣을 위치를 선택하거나 복사/붙여넣기를 해야 합니다.

반면 인라인AI는 문서를 직접 편집하며, 자율적으로 복잡한 표에 내용을 채우거나 각주를 삽입합니다.

분야별 실제 활용 사례

🏛️ 공무원 / 공공기관

  • 정부지원 사업계획서 자동 작성
  • 각종 보고서 초안 생성
  • 정책 문서 작성 자동화

⚖️ 법률 분야

  • 준비서면 초안 작성
  • 각종 법률 서류 자동화
  • 법률 문서 분석 및 요약

🎓 교육 분야

  • 학교생활기록부 작성
  • 창의적 체험활동 기록 자동화
  • 교과 세부능력 특기사항 작성

🔬 연구 분야

  • 영어 논문 한국어 번역
  • 학술 논문 요약
  • 연구 보고서 초안 작성

💼 기업 / 비즈니스

  • IR 자료 제작
  • 제안서 작성
  • 견적서 자동 생성

보안은 안전한가요?

인라인AI의 3중 보안 시스템을 살펴보겠습니다.

🔒 1단계: 문서 외부 전송 차단

편집 중인 문서는 기존 한글 프로그램으로 열리기 때문에 컴퓨터 외부로 전송되지 않습니다.

한컴이나 MS가 여러분의 컴퓨터 속 문서를 볼 수 없는 것과 같은 원리입니다.

🔒 2단계: 철저한 데이터 암호화

채팅에 업로드되는 파일은 모두 CSAP 인증 클라우드에서 AES-256 방식으로 암호화됩니다.

CSAP란? 과학기술정보통신부에서 설정한 클라우드 서비스 정보보호 기준의 충족 여부를 평가하는 인증입니다.

🔒 3단계: AI 학습 금지 정책

  • ✅ 미국 및 대한민국 호스트사만 사용
  • ✅ AI 학습 완전 금지
  • ✅ AI 업체와 데이터 비저장 협약 체결
  • ✅ 중국 등 외부 지역 모델 사용 안 함

최신 업데이트: AI Agent 기능

2025년 7월 업데이트로 완전 자율형 AI Agent 기능이 추가되었습니다.

AI Agent가 하는 일

사용자 지시 → 사내 데이터 자동 수집 → 분석 → 문서 자동 완성

주요 특징:
- 사내 NAS 직접 접근
- 클라우드 저장소 자동 검색
- DMS 등 내부 시스템 연동
- 웹 검색 의존도 제로

웹 검색에 의존하지 않고 기업 내부 자료만으로 고품질 문서를 만들 수 있다는 게 핵심입니다.

자주 묻는 질문 (FAQ)

Q1. ChatGPT와 어떻게 다른가요?

A. 문서를 작성하는 AI라는 점에서 다릅니다. 아래아한글 프로그램에 직접 연동되어 문서를 작성·편집합니다.

Q2. 한컴 AI/MS Copilot과 차이점은?

A. 문서 구조에 따라 내용을 넣을 위치를 자동 선정합니다. 복사/붙여넣기 없이 자율적으로 표에 내용을 채우거나 각주를 넣어줍니다.

Q4. 보안은 안전한가요?

A. 로컬 RAG 시스템으로 데이터를 외부로 전송하지 않으며, CSAP 인증 클라우드에서 AES-256 암호화를 적용합니다.

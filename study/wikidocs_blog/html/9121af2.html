WPF - TextBlock

@kimo
2026년 3월 17일 5:00 오후

WPF(Windows Presentation Foundation)에서 TextBlock은 가장 기본적이면서도 강력한 텍스트 표시 컨트롤입니다.
사용자로부터 입력을 받지 않는 단순 텍스트 출력이 목적일 때 사용하며, 가볍고 성능이 뛰어납니다.
TextBlock의 핵심 기능부터 고급 활용법까지 완벽하게 정리해 드립니다.

1. 기본 사용법가장 기본적인 사용 방법은 Text 속성을 사용하는 것입니다.

<TextBlock Text="안녕하세요, WPF의 세계에 오신 것을 환영합니다!" 
           FontSize="16" 
           Foreground="Blue" />

2. 핵심 속성 (Property)

TextBlock의 외관과 배치를 결정하는 주요 속성들입니다.

속성 설명
Text 표시할 문자열 내용을 설정합니다.
FontSize 글꼴의 크기를 설정합니다.
FontWeight 글꼴의 굵기 (Bold, Normal 등)를 설정합니다.
Foreground 글씨 색상을 설정합니다.
TextAlignment 텍스트 정렬 (Left, Center, Right, Justify)을 제어합니다.
TextWrapping 텍스트가 너비를 넘어갈 때 줄바꿈 여부 (Wrap, NoWrap)를 설정합니다.
TextTrimming 텍스트가 넘칠 때 말줄임표(...) 표시 여부를 설정합니다. (CharacterEllipsis, WordEllipsis)

3. 인라인 요소 (Inlines)

하나의 TextBlock 안에서 부분적으로 다른 스타일을 적용하고 싶을 때 사용합니다. Run, Bold, Italic, LineBreak 등을 조합할 수 있습니다.

<TextBlock FontSize="18" TextWrapping="Wrap">
    <Run Text="이것은 일반 텍스트입니다." />
    <Bold>이 부분은 굵게 표시됩니다.</Bold>
    <Italic Foreground="Red">빨간색 기울임꼴입니다.</Italic>
    <LineBreak/> <Underline>밑줄이 그어진 텍스트</Underline>
    <Hyperlink NavigateUri="https://google.com">클릭 가능한 링크</Hyperlink>
</TextBlock>

4.텍스트 배치 및 레이아웃 제어

TextBlock이 컨테이너 안에서 어떻게 보일지 결정하는 중요한 팁입니다.

줄바꿈과 말줄임표
(Wrapping & Trimming)텍스트가 너무 길 때 UI가 깨지는 것을 방지합니다.

<TextBlock Text="매우 긴 문장이 들어올 경우 여기서 잘리게 됩니다."
           Width="150"
           TextWrapping="NoWrap"
           TextTrimming="CharacterEllipsis" />

줄 간격 제어 (LineHeight)
가독성을 높이기 위해 줄 사이의 간격을 조절할 수 있습니다.

<TextBlock Text="첫 번째 줄&#x0a;두 번째 줄"
           LineHeight="30"
           LineStackingStrategy="BlockLineHeight" />

5. 데이터 바인딩 (Data Binding)

MVVM 패턴이나 동적 데이터를 표시할 때 가장 많이 사용되는 방식입니다

<TextBlock Text="{Binding UserName, FallbackValue='사용자 이름 없음'}"
           FontWeight="Bold" />

6. TextBlock vs Label

  • 많은 입문자가 헷갈려 하는 부분입니다. 핵심 차이점은 다음과 같습니다.
  • TextBlock: Visual 요소로 직접 그려지기 때문에 가볍고 빠릅니다. 텍스트 표시가 주 목적입니다.
  • Label: ContentControl입니다. 다른 컨트롤을 담을 수 있고, Access Key(단축키 _ 사용) 기능을 지원합니다. (예: _Name 입력 시 Alt+N으로 포커스 이동 가능)Tip: 단순 텍스트 표시라면 무조건 TextBlock을 사용하는 것이 성능상 유리합니다.

7. 고급 활용

  • 효과 (Effects)글씨에 그림자를 주거나 외곽선을 넣는 등 시각적 효과를 부여할 수 있습니다.
<TextBlock Text="그림자 효과" FontSize="30">
    <TextBlock.Effect>
        <DropShadowEffect BlurRadius="5" Color="Gray" ShadowDepth="3" />
    </TextBlock.Effect>
</TextBlock>

WPF TextBlock의 진정한 강력함은 단순한 문자열 출력이 아니라, 다양한 인라인(Inline) 요소를 조합해 리치 텍스트(Rich Text)를 구성할 수 있다는 점에 있습니다.

TextBlock.Inlines 컬렉션에 들어갈 수 있는 모든 주요 요소들을 상세히 정리해 드립니다.

8. 텍스트 장식 및 스타일 요소

가장 빈번하게 사용되는 인라인 요소들입니다. 텍스트의 특정 부분만 강조할 때 유용합니다.

<Run>: 가장 기본적인 텍스트 단위입니다. Text 속성을 통해 문자열을 바인딩하거나 직접 입력하며, 개별적으로 Foreground, FontSize 등을 지정할 수 있습니다.

<Bold>: 텍스트를 굵게(FontWeight="Bold") 표시합니다.

<Italic>: 텍스트를 기울임꼴(FontStyle="Italic")로 표시합니다.

<Underline>: 텍스트에 밑줄을 긋습니다.

<Span>: 다른 인라인 요소들을 그룹화하는 컨테이너입니다. 그 자체로는 특별한 스타일이 없지만, 여러 요소를 묶어 한꺼번에 스타일을 적용할 때 사용합니다.

9. 레이아웃 및 구조 요소

텍스트의 흐름을 제어하거나 구조를 잡는 요소입니다.

<LineBreak />: 강제로 줄바꿈을 수행합니다. (HTML의 <br>과 동일)

<Section>: (주로 FlowDocument에서 쓰이지만 TextBlock에서도 논리적 구분을 위해 사용 가능) 블록 수준의 흐름을 구성합니다.

<Figure> / <Floater>: 텍스트 흐름 내에서 특정 위치에 고정되거나 떠 있는 콘텐츠를 배치할 때 사용합니다. (주로 복잡한 문서 레이아웃용)

10. 대화형 및 외부 콘텐츠 요소

단순 출력을 넘어 기능을 부여하거나 컨트롤을 삽입할 때 사용합니다.

<Hyperlink>: 클릭 가능한 링크를 만듭니다. MapsUri를 설정하고 RequestNavigate 이벤트를 처리하여 웹 페이지를 열거나 특정 동작을 수행할 수 있습니다.

<InlineUIContainer>: 가장 강력한 요소 중 하나입니다. 텍스트 흐름 중간에 Button, CheckBox, Image 등 WPF 컨트롤을 직접 삽입할 수 있게 해줍니다.

11. 종합 예제 코드

위의 모든 요소를 하나의 TextBlock에서 구현한 예시입니다.

<TextBlock FontSize="16" TextWrapping="Wrap" LineHeight="25">
    <Run Text="이것은 기본 Run이며," />
    <Bold>이것은 Bold입니다.</Bold>
    <Italic Foreground="DimGray">이것은 Italic이며,</Italic>
    <Underline>이것은 Underline입니다.</Underline>

    <LineBreak/> <Span Foreground="Blue" FontSize="20">
        <Run Text="Span으로 묶인" />
        <Bold>파란색 큰 글자</Bold>
    </Span>

    <LineBreak/>

    <Hyperlink NavigateUri="https://github.com" Foreground="DeepSkyBlue">
        깃허브 방문하기
    </Hyperlink>

    <LineBreak/>

    <Run Text="텍스트 중간에 버튼 넣기:" />
    <InlineUIContainer>
        <Button Content="클릭" Width="50" Height="20" Margin="5,0" Cursor="Hand"/>
    </InlineUIContainer>
    <Run Text="← 이처럼 컨트롤이 텍스트와 함께 흐릅니다." />
</TextBlock>

FastAPI (3/3) : DB(MySQL) 붙여보기

@mattabu
2026년 2월 9일 5:32 오후
Python

이번에는 FastAPI 프레임워크에 MySQL 을 붙여서 동작시켜보겠습니다. 먼저, 윈도우에 MySQL 8.4.x 설치하기 를 참고해서, MySQL을 PC에 설치해줍니다. 그리고, test_db 를 생성하고, test_user 계정도 만들어줍니다. test_user에 test_db를 사용할 수 있는 권한도 줍니다. 이에 내용은 참고 링크에 모두 있어요. TB_ADMIN 이라고 하는 테이블도 만들어줍니다. 샘플 데이터까지 넣어주시고요.

이제, 스토어드 프로시져를 추가로 하나 만들어 줄게요.

DELIMITER $$
CREATE PROCEDURE SP_L_ADMIN ()
BEGIN
    SELECT ADMIN_NO, LOGIN_ID, PASSWD, NICK, EMAIL FROM TB_ADMIN;
END $$
DELIMITER ;

그리고, SP_L_ADMIN을 호출해서 TB_ADMIN의 내용을 조회해 보겠습니다.

mysql> CALL SP_L_ADMIN();

여기까지 DB준비 완료된거에요. 이제, FastAPI 에서 DB 연결부터 조회하는 것까지 해봅시다. ORM 같은 것은 사용하지 않고, 쌩 SQL과 Stored Procedure 를 이용하는 법을 알아봅니다.

먼저 uv add mysql-connector-python 을 실행해서 MySQL 패키지를 설치합니다.

C:\s\fapi>uv add mysql-connector-python
Resolved 16 packages in 171ms
Prepared 1 package in 1.41s
Installed 1 package in 168ms
 + mysql-connector-python==9.3.0

C:\s\fapi>

커넥션풀 관련해서는 https://dev.mysql.com/doc/connector-python/en/connector-python-connection-pooling.html 를 참고할 수 있습니다.

config/config.py을 다음처럼 DB내용설정 내용을 추가해줍니다.

# config/config.py

PORT = 8080

MYSQL_DB_CONFIG = {
    "host": '127.0.0.1',
    "database": 'test_db',
    "user": 'test_user',
    "password": 'test123',
    "pool_size": 10,
    "pool_name": "mysql_pool",
}

fapi 폴더 아래에 model 폴더를 생성하고, 그 안에 mysql_test.py 를 만들어줍니다.

# model/mysql_test.py

from mysql.connector import Error
import mysql.connector
from config import config


def list_admin():
    with mysql.connector.connect(**config.MYSQL_DB_CONFIG) as conn:
        cur = conn.cursor()
        try:
            cur.execute("SELECT * FROM TB_ADMIN")
            results = cur.fetchall()
        except Error as err:
            print(f'Error querying: {err}')
            results = False
    return results

이제, controller 폴더 아래에 admins.py 를 만들겠습니다.

# controller/admins.py

from fastapi import APIRouter
from model import mysql_test

router = APIRouter(
    prefix="/admins",
    tags=["admins"],
    responses={404: {"description": "Not found"}},
)

@router.get("/list")
def list_admin():
    results = mysql_test.list_admin()
    return results

app.py 에도 admins 를 추가해주세요.

# app.py

from fastapi import FastAPI
from controller import items, users, admins

app = FastAPI()

app.include_router(items.router)
app.include_router(users.router)
app.include_router(admins.router)

@app.get("/")
def read_root():
    return {"Hello": "World"}

웹브라우저에서 http://localhost:8080/admins/list 를 호출해봅니다.

이번에는 mysql_test.py 의 list_admin() 함수를 아래처럼 바꾸어본다. 위에서는 일반 SQL을 실행했다면, 이번에는 Stored Procedure 를 실행해보는 것입니다.

def list_admin():
    results = []
    with mysql.connector.connect(**config.MYSQL_DB_CONFIG) as conn:
        cur = conn.cursor()
        try:
            cur.callproc("SP_L_ADMIN")
            for result in cur.stored_results():
                results.append(result.fetchall())
        except Error as err:
            print(f'Error querying: {err}')
            results = False
    return results

그리고, 또 다시 http://localhost:8080/admins/list 를 호출해보세요.

여기까지 FastAPI 에 MySQL 연결해서 사용해보는 것까지 알아보았습니다. 더 해볼 것들이 있기는 하지만, 이정도만 해도 전체적인 틀을 보는데이는 무리가 없을 것이고 생각합니다. 나머지는 여러분들이 추가적으로 더 공부를 해서 더해자가면 될 것 같아요.

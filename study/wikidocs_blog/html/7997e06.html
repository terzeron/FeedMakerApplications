Python 에서 가장 핫한 웹프레임워크는 바로 FastAPI 입니다. Django, Flask 도 유명하긴 하지만, 딱히 마음에 들지 않았습니다. 이름에서도 알 수 있드시 성능도 2배 정도 더 빠르다고 하네요. 그리고, 더 간결하고 편해요. 저는 주로 API서버로 사용하고 있어서 딱입니다. 물론 필요한 경우에 Jinja 를 이용해서 페이지를 만들어낼 수도 있어요.

본격적으로 시작해봅시다. 처음으로 해야 할 일은 uv 를 설치하는 것입니다. uv 에 대해서는 uv : 엄청 빠르고 편리한 Python 프로젝트, 패키지 관리 프로그램에서 추가적으로 참고해주세요.

저는 윈도우 환경므로 아래처럼 설치를 이미 했습니다. 여러분도 설치가 안되어 있으면, 설치해주세요.

> powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

Mac 이나 Linux 환경이시면, 다음처럼 해주시면 됩니다.

$ curl -LsSf https://astral.sh/uv/install.sh | sh

위와 같이 설치가 진행이 잘 되고, pyproject.toml 파일을 열어보면 fastapi 와 uvicorn 이 dependencies 항목에 추가된 것도 확인할 수 있어요.

[project]
name = "fapi"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.13"
dependencies = [
    "fastapi>=0.115.12",
    "uvicorn>=0.34.1",
]

main.py 파일을 열어서, 다음을 입력해줍니다.

# main.py

import uvicorn

def main():
    uvicorn.run(app="app:app", host="0.0.0.0", port=8080, reload=True)

if __name__ == "__main__":
    main()

위에서 보시면, app="app:app" 이라고 되어 있는 부분이 보일거에요. app.py 파일에 있는 app 객체를 참고하라는 의미입니다. 그리고, reload=True 항목은 프로그램 소스를 수정하면 자동으로 재시작할 수 있는 옵션이에요. 개발단계에서는 편리하겠지요. 당연하겠지만, 운영환경에서는 False로 지정해주어야 합니다.

다음으로, app.py을 만들어서 다음처럼 입력해줍니다. 여기에는 실제 FastAPI 서버에서 작동할 코드를 넣습니다.

# app.py

from typing import Union
from fastapi import FastAPI

app = FastAPI()
@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}

uv run main.py 라고 입력해서 실행해봅니다. 아래처럼 나오면 실행이 잘 된거에요.

http://localhost:8080/docs 라고 입력하면, Swagger UI 로도 API 문서를 볼 수 있습니다. 좋구만요~!

이번에는 포트 정보를 설정파일(config.py)로 빼서 사용해봅시다. fapi 폴더 아래에 config 폴더를 만들고, 그 안에 config.py 를 다음과 같이 작성합니다.

# config/config.py

PORT = 8080

간단하지요? 나중에 이 파일에 DB연결정보도 넣을 예정이에요.

main.py 파일을 수정해줍니다.

# main.py

import uvicorn
from config import config

def main():
    uvicorn.run(app="app:app", host="0.0.0.0", port=config.PORT, reload=True)

if __name__ == "__main__":
    main()

이 상태에서도 http://localhost:8080 로 접속이 잘되면 성공~!!!

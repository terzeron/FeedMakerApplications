[NumPy와 행렬 연산]

  1. NumPy 라이브러리의 주요 특징으로 옳지 않은 것은?
    ① 파이썬의 기본 기능보다 수치 연산 속도가 빠르다.
    ② C언어로 구현된 함수를 사용하여 연산을 수행한다. --> Numpy의 핵심 연산부분은 C언어로 구현되어 있음.
    ③ 주로 비정형 텍스트 데이터의 합산과 병합에만 특화되어 있다.
    ④ 거의 모든 머신러닝 및 딥러닝 라이브러리에서 사용된다.

  2. NumPy에서 모든 원소가 0인 3x4 행렬을 생성하는 함수는?
    ① np.array((3,4))
    ② np.zeros((3,4))
    ③ np.ones((3,4))
    ④ np.arange(12)

  3. np.arange(1, 10, 2)를 실행했을 때 생성되는 배열의 모습은?
    ① [1, 2, 3, 4, 5, 6, 7, 8, 9]
    ② [1, 3, 5, 7, 9] --> np.arange(시작, 끝, 간격) - 끝값은 포함 X
    ③ [2, 4, 6, 8]
    ④ [1, 10, 2]

  4. 두 벡터의 방향 유사도를 측정할 때 주로 활용되는 연산은?
    ① 요소별 합(Addition)
    ② 내적(Dot Product)
    ③ 노름(Norm)
    ④ 전치(Transpose)

  5. NumPy에서 벡터의 크기나 거리 개념을 계산할 때 사용하는 함수는?
    ① np.dot()
    ② np.linalg.norm() --> Norm = 벡터의 크기/거리
    ③ np.transpose()
    ④ np.random.randn()

  6. 두 행렬 A와 B의 행렬 곱(Matrix Multiplication)을 수행하는 올바른 방법은? (2개 선택)
    ① A * B
    ② np.matmul(A, B)
    ③ A @ B
    ④ A.T @ B.T

  7. 행렬의 행과 열을 바꾸는 연산을 무엇이라 하는가?
    ① 역행렬
    ② 전치 행렬(Transpose)
    ③ 영행렬
    ④ 단위 행렬

  8. np.random.randn(m, n) 함수는 어떤 배열을 생성하는가?
    ① 0과 1 사이의 균등분포 난수 배열
    ② 표준정규분포를 따르는 난수 배열
    ③ m부터 n까지의 정수 배열
    ④모든 원소가 n인 크기 m의 배열

Pandas 데이터 처리]

  1. Pandas 라이브러리에 대한 설명으로 옳은 것은?
    ① 오직 3차원 이상의 고차원 텐서 연산만을 위해 존재한다.
    ② 테이블 형태의 정형 데이터 처리에 특화되어 있다.
    ③ 수치 연산 속도가 NumPy보다 항상 빠르다.
    ④ 텍스트 전처리 중 형태소 분석만을 담당한다.

  2. Pandas에서 제공하는 1차원 데이터 구조의 명칭은?
    ① DataFrame
    ② Series --> 1차원
    ③ Matrix
    ④ Array

  3. DataFrame에 대한 설명으로 틀린 것은?
    ① 2차원 구조를 가진다.
    ② 여러 개의 Series가 결합된 형태이다.
    ③ 행(Row)과 열(Column)로 구성된다.
    ④ 데이터의 순서가 없으므로 인덱싱이 불가능하다.

  4. 데이터프레임 df의 행과 열의 크기를 확인하고 싶을 때 사용하는 속성은?
    ① df.columns
    ② df.info()
    ③ df.shape --> 모양/크기
    ④ df.describe()

  5. 데이터프레임의 컬럼 이름들만 모아서 확인하고 싶을 때 사용하는 속성은?
    ① df.head()
    ② df.columns
    ③ df.values
    ④ df.index

  6. 데이터프레임의 구조, 결측치 여부, 데이터 타입 등을 요약해서 보여주는 함수는?
    ① df.describe()
    ② df.info()
    ③ df.summary()
    ④ df.check()

  7. 수치형 데이터들의 평균, 표준편차, 사분위수 등 통계 정보를 한눈에 보여주는 함수는?
    ① df.describe() --> 통계 정보 제공
    ② df.mean()
    ③ df.info()
    ④ df.stat()

[종합 및 텍스트 데이터 분석 기초]

  1. 다음 중 정형 데이터 처리 및 병합에 가장 적합한 도구는?
    ① NumPy
    ② Pandas
    ③ Matplotlib
    ④ Scikit-learn

  2. 다음 중 NumPy 배열 초기화 함수가 아닌 것은?
    ① np.ones()
    ② np.zeros()
    ③ np.array()
    ④ np.dataframe()

  3. 벡터 v1 = [1, 2, 3]일 때, np.dot(v1, v1)의 결과값은?
    ① 6
    ② 14
    ③ 9
    ④ 3.74

  4. 데이터 분석의 일반적인 순서로 옳은 것은?
    ① 데이터 수집 → 전처리 → 데이터 분석/변환 → 결과 도출
    ② 결과 도출 → 전처리 → 데이터 수집 → 데이터 분석
    ③ 데이터 분석 → 데이터 수집 → 결과 도출 → 전처리
    ④ 전처리 → 결과 도출 → 데이터 수집 → 데이터 분석

  5. RAG(Retrieval Augmented Generation) 기술과 관련하여 데이터 분석 교재에서 다루는 핵심 기법의 순서가 아닌 것은?
    ① NumPy와 행렬
    ② Pandas 데이터 처리
    ③ 텍스트 전처리 핵심 기법
    ④ 이미지 렌더링 및 필터링

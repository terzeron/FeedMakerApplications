나만의 MCP 서버 만들기 with 커서 AI - 리뷰

@kimo
2025년 9월 2일 1:29 오후


요즘 IT 사업군에서 AI , MCP 등의 키워드가 트렌드 되어 가고 있습니다.
MCP에 대해서 항상 궁금중이 많았는데 이 책을 읽고 그 궁금증이 많이 해결되었습니다.
이젠 AI는 하나의 솔루션이 아닌 다양한 솔루션이고 이걸 이어서 만들 수 있게 MCP 라는
프로토콜이 나왔으며 MCP를 만들기 위한 호스트로 커서와 클로드 데스크톱을 사용하고 있습니다.

책에서는 실습 환경에 대해서 자세히 설명을 해줍니다. 커서라던지 클로드 데스크톱을 IDE(호스트)를 사용하는 방법
그리고 다양한 API 키를 이용한 데이터를 가져와서 AI를 사용하기도 합니다.

질문? Chat GPT , 클로드 및 제미나이가 있는데 왜 커서를 설치하고 클로드 데스크탑을 설치해서 사용하고 있나요?
- 이 내용에 대해서 친구랑 이야기 한적이 있다. 우리가 어떠한 문의를 할때 전화로 서비스 센터에 연락을 할 때가 있는데 이 예시를 우리가 웹브라우저에서 GPT나 클로드를 사용하는 것과 같다 하지만 클로드 데스크탑 혹은 커서를 이용한다라는 것은 콜 센터에 전화를 한게 아니라 A/S 기사 아저씨가 직접와서 도와준다라고 생각하면 좋을 것 같다.

간단히 이 책에서는 ChatGPT의 API 키를 이용한 내 컴퓨터에 여러 솔루션을 제어 할 수 있는 GPT를 만든다고 생각하면 좋을 것 같다.

  • 이 책은 길벗에서 책 후원을 받아 작성된 리뷰 글입니다.
